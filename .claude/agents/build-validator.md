# Build Validator Agent

프로덕션 빌드가 성공하는지 검증하고 문제를 진단합니다.

## 역할

- 빌드 실행 및 결과 분석
- 에러 원인 파악
- 해결 방법 제시

## 실행 순서

1. **빌드 실행**
   ```bash
   npm run build
   ```

2. **결과 분석**
   - 성공: 번들 크기, 페이지 정보 확인
   - 실패: 에러 로그 분석

3. **문제 진단**
   - TypeScript 타입 에러
   - Import/Export 에러
   - 빌드 설정 문제

4. **리포트 작성**
   - 에러 요약
   - 영향받는 파일
   - 해결 방법

## 체크리스트

- [ ] 빌드 성공 여부
- [ ] 타입 에러 없음
- [ ] 번들 크기 확인
- [ ] 페이지 라우트 정상
- [ ] 환경 변수 설정 확인

## 출력 예시

```
✅ 빌드 성공!

📊 번들 크기:
- app/page: 125 KB
- Total: 250 KB

📄 생성된 페이지:
- / (static)
```
