## 💡 소개

원격제어 솔루션 전문 기업에서 macOS 네이티브 애플리케이션을 개발하고 있습니다.

**입사 후 지속적으로 3개 제품(RemoteView, RemoteCall, RemoteMeeting)을 동시 개발 및 유지보수**하며,
신규 기능 개발부터 고객사 솔루션 납품까지 전 과정을 경험했습니다.

특히 **RV RXHost** 프로젝트에서는 5개 프로세스로 구성된 멀티프로세스 아키텍처를 기반으로
실시간 화면 캡처, 원격 제어, 네트워크 통신 등 핵심 모듈을 개발했으며,
**WebRTC 기반 음성채팅, Canvas 기반 실시간 그리기, H.265 코덱** 등 신규 기능을 자체 개발했습니다.

## 🛠️ 기술 스택

### 프로그래밍 언어
- **Objective-C** (주력) - macOS 네이티브 개발, 851개 헤더 / 575개 구현 파일 규모 프로젝트 경험
- **Objective-C++** - 성능 크리티컬 영역 (네트워크 래퍼, 스크린 캡처 엔진)
- **C/C++** - 시스템 레벨 모듈, 사내 기반 기술 적용
- **Swift** - 최신 macOS 기능 적용

### macOS 시스템 프로그래밍
- **XPC (Inter-Process Communication)** - 5개 독립 프로세스 간 통신 구현
- **Screen Capture** - Xenc SDK 통합, H.264/H.265 인코딩, 멀티 모니터 지원
- **Keyboard/Mouse Control** - PAE SDK 통합, CGEvent 기반 이벤트 주입
- **Accessibility API** - 시스템 제어 권한 관리
- **IOKit** - 디스플레이 정보 조회, 전원 관리
- **Core Graphics/Quartz** - 화면 캡처, 이미지 처리
- **launchd** - 백그라운드 데몬 서비스 구현

### 네트워크 & 프로토콜
- **MQTT** - 비동기 메시지 브로커 기반 통신
- **PNS Protocol** - 70+ 메시지 타입 설계 및 구현 (Screen, Control, Status, FTP, Clipboard 등)
- **JSON** - 메시지 직렬화/역직렬화
- **WebRTC** - 실시간 음성채팅 구현
- **TCP/IP, UDP** - 원격제어 프로토콜 구현

### 빌드 & 배포
- **Xcode Workspace** - 5개 프로젝트 통합 관리
- **Jenkins CI/CD** - 자동 빌드 및 배포 파이프라인
- **Git/GitLab** - 브랜치 전략 수립 (SaaS / Solution / Custom)
- **macOS App 서명** - 코드 사이닝 및 배포 자동화
